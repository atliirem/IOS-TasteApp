# IOS-TasteApp
# 🍽️ TasteApp – iOS Food Discovery & Recipe App

TasteApp is a modern iOS application built with **SwiftUI** that allows users to explore meals, search by category, view detailed recipes, and create their own meals with Firebase integration.

---

## 🚀 Features

* 🔍 **Search & Filter**

  * Search meals by name or category
  * Dynamic filtering system

* 🍔 **Meal Listing**

  * API-based meal fetching (TheMealDB)
  * Smooth scrolling with LazyVStack

* 📄 **Meal Detail Page**

  * Large image preview
  * Category, area, instructions
  * Ingredients list

* ❤️ **Favorites System**

  * Add/remove favorite meals
  * Stored locally using UserDefaults

* 🔐 **Authentication**

  * Firebase Authentication (Email & Password)
  * Login / Logout system

* ☁️ **Create Your Own Meal**

  * Add custom recipes
  * Stored in Firebase Firestore
  * Modal-based form UI

* 🖼️ **Image Handling**

  * Async image loading
  * Image caching for performance

---

## 🛠️ Tech Stack

* **SwiftUI**
* **Firebase Auth**
* **Firebase Firestore**
* **URLSession (Async/Await)**
* **MVVM-inspired structure**
* **UserDefaults (Local Storage)**

## ⚙️ Setup

1. Clone the repository

```bash
git clone https://github.com/atliirem/IOS-TasteApp.git
```

2. Open in Xcode

3. Add your Firebase configuration file:

* `GoogleService-Info.plist`

4. Run the project
